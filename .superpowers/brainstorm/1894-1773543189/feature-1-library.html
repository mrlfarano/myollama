How It Works

1

Curated Catalog + Search

A bundled JSON catalog of ~50 popular models with descriptions, tags, and size ranges. Search filters against this catalog. The "Pull by name" bar at the bottom handles anything not in the catalog.

2

Tag Selection

Clicking "Tags" or "Pull Model" expands to show available tags/quantizations (e.g., :4b, :1.7b, :latest, :q4_K_M). User picks the specific variant to pull.

3

Live Pull Progress

Model downloads stream progress via the Ollama /api/pull endpoint (SSE). Progress bar, speed, and ETA shown on the card. Multiple concurrent pulls supported.
