MyOllama — Feature Scope

Click to select features for v1. Unselected features move to "later".

Model Management

1

Model Library Browser

Browse available models from the Ollama registry, search, filter by size/family/quantization, and pull new models with progress tracking.

2

Installed Models Dashboard

View all installed models with size, family, quantization, parameter count, last used date. Delete, duplicate, or update models.

3

Modelfile Editor

Create and edit Modelfiles visually — set system prompts, parameters, template overrides. Build custom model variants.

Inference & Chat

4

Chat Playground

Full chat interface with streaming responses, conversation history, model switching mid-conversation, and markdown rendering.

5

Parameter Tuning Panel

Adjust temperature, top_p, top_k, repeat penalty, context window size, and other inference parameters with real-time sliders.

6

Side-by-Side Model Comparison

Send the same prompt to 2+ models simultaneously, compare outputs, response times, and token usage side by side.

7

Prompt Templates & Library

Save, organize, and reuse system prompts and prompt templates. Share across conversations.

Monitoring & System

8

Resource Monitor

Live GPU/CPU/RAM usage, VRAM allocation per model, inference speed (tokens/sec), and system health dashboard.

9

Request Logs & History

Log all API requests with timing, token counts, parameters used. Filter, search, and replay past requests.

10

Multi-Endpoint Switching

Save multiple Ollama server URLs (e.g., Orin, desktop, cloud) and switch between them from a dropdown. Health check indicators.

Advanced

11

API Explorer / Docs

Interactive API documentation — test Ollama endpoints directly, see request/response schemas, generate code snippets for curl/Python/JS.

12

Embedding Tools

Generate and visualize embeddings, test similarity search, manage embedding models separately from chat models.

13

Vision / Multimodal Support

Upload images to vision-capable models (LLaVA, etc.), drag-and-drop image input in the chat interface.

14

Batch Inference

Run a prompt against multiple inputs (CSV/JSON), collect results in a table. Useful for evaluation and data processing.
